Designer

Imani Greenidge, Brooklyn High School for Law and Technology Class of 2018 former AP Chemistry student of Ms. Lyons. Check out his portfolio

Purpose

I made this website to give both AP Chemistry and Regents Chemistry Students of Ms. Lyons a place to go to get resources they made need to be successful in class. You can get all the resources you need for class through this website without even having to bother Ms. Lyons. This website should be used as a tool to help chemistry students succeed.

How

Using my knowledge of coding and web design. Coding languages such as HTML, CSS and Javascript I decided to make this website for Ms.Lyons' chemistry classes (AP and Regents). I used a library named Materialize Css to create majority of the layout.
